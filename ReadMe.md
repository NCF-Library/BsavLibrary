# Bsav Library

Botw Save Library for reading and writing BOTW save files (.sav)

### Install

[![NuGet](https://img.shields.io/nuget/v/BsavLibrary.svg)](https://www.nuget.org/packages/BsavLibrary) [![NuGet](https://img.shields.io/nuget/dt/BsavLibrary.svg)](https://www.nuget.org/packages/BsavLibrary)

#### NuGet
```powershell
Install-Package BsavLibrary
```

#### Build From Source
```batch
git clone https://github.com/NCF-Library/BsavLibrary.git
dotnet build BsavLibrary
```
